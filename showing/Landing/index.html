О проекте

Разрабатываемый проект: сайт по заказу домашних тортов «От Павловых».

Проблема: при создании домашнего бизнеса рано или поздно многие сталкиваются с рядом проблем:

  1. Малая известность компании;
  2. Неудобный формат взаимодействия с клиентами;
  3. Неудобный формат работы с поступающей информацией о заказах и др.

Цель работы: создать сайт по заказу домашних тортов, который будет отображать всю необходимую для совершения целевого действия информацию пользователю, повысит удобство сбора требований к заказу, поспособствует увеличению числа клиентов.
